Sintaxis CSS

Un conjunto de reglas CSS consta de un selector y un bloque de declaración:

  • El contenido que es lo "importante", el texto en las etiquetas de tezxto, la imagen en las etiquetas de imagen, etc...
    • El padding es la distancia que existe entre el contenido y el borde de la caja
    • El borde que es el elemento que marca la división entre el elemento y el resto de los elementos de la página
    • El margen que es la distancia entre el elemento y el resto de los elementos de la página
